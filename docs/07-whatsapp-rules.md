# Regras de Contato via WhatsApp

## Objetivo
Transformar a landing page em um canal de conversão simples e rápido.

## Configuração base
Cada loteamento deve possuir:
- nome comercial
- número de WhatsApp em formato internacional
- mensagem padrão opcional

## Botão flutuante
Obrigatório no MVP.

### Regras
- exibir em toda a landing page
- abrir `wa.me`
- usar mensagem padrão do loteamento
- respeitar mobile e desktop

## CTA ao clicar no lote
Exibir no painel do lote selecionado:

- botão de tirar dúvidas
- botão de interesse/aquisição

## Mensagem sugerida com contexto
`Olá! Tenho interesse no lote {quadra}-{numero}, com área de {area}m², no loteamento {nome}. Pode me passar mais detalhes?`

## Observações
- o número deve ser do loteamento/empresa, não do visitante
- no MVP não é necessário integrar API oficial do WhatsApp
- basta abrir o link com mensagem pré-preenchida

## Futuro
- distribuição por corretor
- múltiplos números por loteamento
- rastreamento de origem do clique
- integração com CRM
