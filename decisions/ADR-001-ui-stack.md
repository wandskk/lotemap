# ADR-001 — UI Stack

## Status
Aceito

## Contexto
É necessário padronizar a camada visual do produto, equilibrando velocidade de desenvolvimento e consistência.

## Decisão
Adotar **shadcn/ui + Tailwind CSS**.

## Consequências
### Positivas
- acelera desenvolvimento
- componentes reutilizáveis
- ótimo encaixe com Next.js
- excelente composição para dashboard e landing page

### Negativas
- dependência prática de Tailwind
- exige disciplina para não gerar classes desorganizadas
