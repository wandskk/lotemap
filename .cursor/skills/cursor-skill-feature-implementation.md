# Skill — Feature Implementation

## Objetivo
Implementar uma feature do LoteMap de forma segura e incremental.

## Fluxo
1. ler a spec
2. localizar entidades e rotas envolvidas
3. definir schemas Zod
4. implementar serviço
5. implementar UI
6. proteger com autorização
7. validar critérios de aceite

## Princípios
- alterar pouco por vez
- não quebrar o domínio multi-tenant
- manter dados públicos e internos separados
