# Skill — Product Planning

Use esta skill quando estiver refinando o LoteMap no Cursor.

## Objetivo
Quebrar o produto em entregas pequenas e implementáveis.

## Regras
- respeitar o escopo do MVP
- não adicionar features fora do escopo sem registrar decisão
- sempre produzir:
  - objetivo
  - arquivos afetados
  - passos técnicos
  - critérios de aceite
  - riscos

## Saída esperada
- plano curto
- checklist
- observações de risco
